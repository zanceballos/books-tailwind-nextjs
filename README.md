# 🦉 Bookify Owl

[![Live Demo](https://img.shields.io/badge/Live%20Demo-View%20Website-purple?style=for-the-badge&logo=netlify)](https://bookify-owl.netlify.app/)
[![GitHub Repo](https://img.shields.io/badge/GitHub-View%20Code-black?style=for-the-badge&logo=github)](https://github.com/zanceballos/books-tailwind-nextjs)

**Bookify Owl** is a modern, cinematic web application designed to help users discover, track, and manage their reading journeys. Powered by Next.js and the Google Books API, it features an AI-enhanced search engine and a beautifully responsive UI.

## ✨ Features

* **Cinematic Discovery:** A "Netflix-style" layout for exploring trending books, new releases, and personalized recommendations.
* **AI-Powered Search:** Integrated with **Groq AI** to understand complex search queries and provide hyper-relevant book suggestions.
* **Personalized Bookshelves:** Users can create custom shelves, track reading progress, and save their favorite books.
* **Authentication & Security:** Secure user login, registration, and password management powered by Firebase Auth.
* **Real-time Database:** Stores user reviews, favorites, and library data instantly using Google Firestore.
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop using Chakra UI.

## 🛠️ Tech Stack

**Frontend:**
* [Next.js](https://nextjs.org/) (React Framework)
* [Chakra UI](https://chakra-ui.com/) (Component Library & Styling)

**Backend & APIs:**
* Next.js Serverless API Routes
* [Firebase](https://firebase.google.com/) (Authentication & Firestore Database)
* [Google Books API](https://developers.google.com/books) (Book Data)
* [Groq API](https://groq.com/) (AI Search & Summarization)
